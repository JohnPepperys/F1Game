Описание: в проекте GameClass собраны классы содержащие игровую механику.
	ok. Класс команды,
		- название, описание, год создания. Технические плюшки, которые (в дальнейшем) будут учитываться при квалификации и гонке.

	ok. Класс пилота,
		- имя, номер, команда, возраст, опыт - количество гонок (в дальнейшем будут вылетать менее опытные гонщики)

	Класс трассы,
		- времена, название, описание, городская/обычная (отразиться на вероятности аварии)
	
	Класс квалификации, 
		- класс действия. На вход список пилотов и трасса, на выходе список с позициями пилотов. Можно статик.

	Класс гонки.
		- класс действия. На вход список пилотов и трасса, на выходе список с позициями пилотов. Можно статик.

	Класс чемпионата. 
		- класс действия. На вход результаты гонки, в процессе сохраняем турнирную таблицу.

	Остальные проекты только отображают игровой процесс и передают кнопки управления внутрь.
